# Kernel -> Rust full-port plan (autonomous, one component per change)

Directive: port the ENTIRE kernel to Rust, one change at a time, single agent, continuous, no confirmation. Full C backup on NAS. Strangler pattern: keep <fn>_c, add <fn>_rs, route under -DRUST_<NAME>, boot-time differential rs==c, verify on throwaway, orchestrator checkpoint-folds to golden.
